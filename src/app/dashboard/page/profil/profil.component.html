Profil

Votre nom ne peut être modifier.
Votre email ne peut être modifier.
Vous pouvez saisir un nouveau mot de passe. Le mot de passe doit être supérieur à 2 caractères.

Vous pouvez modifier votre profil selon votre convenance

Vous ne pourrez modifier seulement que votre Mot de passe et votre Photo
Les autres informations telles que le Nom et l'Email ne peuvent être modifiées parce qu'elles permettent de vous identifier de façon unique.
Si vous constatez une erreur sur vos informations ou si vous voulez modifier votre nom ou votre email, merci de contacter la gouvernante qui se chargera de le faire.
Le Mot de passe doit contenir au minimum 3 caractères, faute de quoi, un message vous est envoyé: Le mot de passe doit être supérieur à 2 caractères !
Si vous ne choisissez aucune Photo ou n'entrez aucun Mot de passe modifié, le bouton de soumission n'a aucun effet.
La photo de profil que vous choisirez devra être inférieur à 1 Mo (Méga Octet) en terme de poid, parce que l'image de profil ne s'affiche que de façon très petite.
Si vous choisissez une image qui est volumineuse, un message d'erreur vous sera envoyé, vous indiquant que L'image que vous avez choisit est très grande. Merci de choisir une image dont le poid est inférieur à 1 Mo !
